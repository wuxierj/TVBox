# 影视仓/TVBox 接口合集

> 2026-01-17 更新 | 欢迎PR补充/修正

## 使用说明
1. 复制下方任意接口地址 → 进入影视仓/TVBox设置 → 配置地址 → 粘贴 → 确定即可。
2. 若提示“配置失败”请切换其他线路，或检查是否被墙。
3. 接口有效性会随时间波动，建议收藏本仓库以便获取最新列表。
